Пример запуска:
TagCloud.exe -f Data/Lorem2.txt -c -i -s 3 -w 1500 -h 750 -b Green

Формат запуска:
TagCloud.exe -f <Имя файла>

Ключи:
-f --file
	Параметры: <Имя файла>
	Описание: Выбирает файл для построения по нему облака тегов
-g --gen
	Параметры: <Имя генератора>
	Описание: Выбор генератора для построения облака тегов
	Доступные параметры: 
		sg - Спираль с шагом по углу
		ssg - Спираль с шагом по дуге
		rssg - Спираль с большим шагом по дуге и радиусом
	Начальное значение: ssg
-w --width
	Параметры: <Число>
	Описание: Ширина итогового изображения
	Начальное значение: 1000
-h --height
	Параметры: <Число>
	Описание: Высота итогового изображения
	Начальное значение: 600
-l --log
	Описание: Выключение логарифмической шкалы (Использовать только на маленьких текстах!)
-s --multipleSize
	Параметры: <Число>
	Описание: Коэффициент линейного увеличения шрифта
	Начальное значение: 5
-i --ignoreCase
	Описание: Игнорирование регистра текста
-c --clear
	Описание: удаляет из текста все символы помимо букв
-n --fontFamilyName
	Параметры: <Font name>
	Описание: задаёт название шрифта
-b --brush
	Параметры: <Название>
	Описание: Задаёт цвет текста
-o --font
	Параметры: <Название>
	Описание: Задаёт цвет фона
